🌤️ Weather — Python Desktop Weather App

Modern desktop app built with **Python**, **CustomTkinter**, and **OpenWeather API**.
A clean UI focused on minimalism, real-time data, favourite locations all in one simple desktop app.

✨ Features
- 🔍 **Search any city**
- ⭐ **Favourite places** (stored in local `favourites.json` file)
- 🌡️ **Dynamic temperature** color (cold → warm gradient)
- 🌅 **Sunrise and sunset times**
- 🌧️ **Rain precipitation**
- 💧 **Humidity and wind speed**
- 🌪️ **Wind speed and angle**
- 🔁 **Refresh button to update all widgets**
- 📁 **Smart path handling** (works in normal Python file and compiled `.exe`)
- 🖼️ **Custom icons** + responsive UI built with CustomTkinter
- 🏙️ **Default city** choose a city that shows on start up
- 🌦️ **Forecast**
---

🚀 Installation (Executable)
1. Go to **Releases**.
2. Download the lastest `.zip`.
3. Extract it anywhere you like.
4. Run the application (`Weather.exe`)

🛠️ Technologies Used
- Python
- CustomTkinter
- Pillow (PIL)
- Requests
- dotenv
- OpenWeather API
